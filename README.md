# LOGIN_SIGNUP_PAGE-EXPRESSJS

***Getting Started***

This is a user login/signup code using Node.js, Express, MYSQL, and JWT.

***Prerequisites***

You need to have installed Node.js, NPM and MYSQL in your System.

***Installing***

```npm install```

***To Run***
    
    ```nodemon login-signup.js```
   
   Code is Running on
   
   ```http://localhost:3000/```
